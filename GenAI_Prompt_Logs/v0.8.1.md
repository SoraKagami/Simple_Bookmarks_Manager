## Summary of changes

### Privacy text update

Changed the About page Privacy wording to be clearer and firmer:

> SBM is designed to run purely locally. It does not send bookmark data, telemetry, pings, or any other usage data to remote services.

This removes “intentionally” and makes the privacy expectation explicit.

### About page text moved into English locale

Moved the About page’s user-facing copy into `_locales/en/messages.json`, including:

- About page title
- Project heading

- GitHub label
- Design goals heading
- Design goal bullet points
- Privacy heading
- Privacy description

The About page now uses `data-i18n` attributes, so these strings can be translated through the normal locale workflow later.

### Added About page localization script

Added `about.js`, which:

- Loads the stored SBM UI language setting.
- Applies the configured locale to `about.html`.
- Applies SBM’s configured UI font family, font size, and line spacing so the About page better matches the rest of the app.
- Keeps static English fallback text visible if locale loading fails.
