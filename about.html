Simple Bookmarks Manager

A simple local bookmarks manager for Chromium browsers, inspired by Firefox Places, Firefox’s bookmark management system.

Project

Simple Bookmarks Manager focuses on local bookmark organization, efficient folder browsing, and a familiar desktop-style management workflow inside Chromium-based browsers.

GitHub: github.com/SoraKagami/Simple_Bookmarks_Manager

Design goals

  • Fast local bookmark browsing and editing.
  • Clear controls without remote services or accounts.
  • Practical compatibility across Chromium-based browsers.

Privacy

SBM is designed to run purely locally. It does not send bookmark data, telemetry, pings, or any other usage data to remote services.
