Simple Bookmarks Manager

A practical guide to using SBM for local bookmark browsing, editing, sorting, and organization.

Quick start

Open SBM from the browser toolbar button. The manager opens as a normal extension page and works directly with Chromium's existing bookmark tree.

  • Library pane: browse bookmark folders and expand or collapse subfolders.
  • Folder Contents pane: view, select, sort, open, drag, and manage items in the selected folder.
  • Details pane: inspect and edit the selected bookmark or folder.

Basic bookmark management

  • Use New Folder and New Bookmark to add items to the current folder.
  • Right-click folders or bookmarks for actions such as rename, edit, delete, cut, copy, paste, sort, and open.
  • Use drag-and-drop to reorder items or move bookmarks and folders into another folder.
  • Use the Details pane to change bookmark names, URLs, parent folders, and supported advanced fields.
  • Use F2 to rename the active folder or edit the active bookmark.
  • Middle-click a bookmark in the Folder Contents pane to open it in a new tab.

Search and sorting

  • Use the search box to search bookmarks by name or URL.
  • Enable Limit Search to Current Folder & Subfolders to search only inside the current folder tree.
  • Use the visual sort dropdown or column headers for temporary sorting in the Folder Contents pane.
  • Use right-click permanent sorting when you want Chromium's saved bookmark order changed.

Separators:

SBM supports a Chromium-compatible separator convention. A separator is stored as a bookmark with the title ——— and URL about:blank. SBM renders that convention as a visual divider.

Keyboard shortcuts

  • Arrow keys: move through the active pane.
  • Enter: open a bookmark or navigate into a folder.
  • Backspace: move up one folder level when available.
  • Ctrl+F: focus the search box.
  • F2: rename the active folder or edit the active bookmark.
  • Delete: delete the selected item when keyboard deletion is enabled.
  • Escape: close open SBM dialogs or menus.

Pinning Simple Bookmarks Manager for easy access

Pinning Simple Bookmarks Manager (SBM) to the toolbar allows faster and easier access to the extension.

Once SBM is installed in your Chromium-based browser, you can pin it to the toolbar in one of the following ways:

1. Pin via the Extensions menu

  1. Click the Extensions button, usually located to the right of the address bar.
  2. Find Simple Bookmarks Manager in the list.
  3. Click the pin icon next to Simple Bookmarks Manager.

This will pin SBM to the toolbar. If SBM is already pinned, clicking the pin icon will unpin it.

2. Pin via the Manage Extensions page

  1. Click the Extensions button, usually located to the right of the address bar.
  2. Click Manage Extensions.

Alternatively:

  1. Click the Chromium menu button.
  2. Select Extensions.
  3. Click Manage Extensions.

Once the Extensions page is open:

  1. Find Simple Bookmarks Manager in the list.
  2. Click Details.
  3. Scroll down to Pin to toolbar.
  4. Toggle the option on.

Options reference

The Options page stores persistent settings in chrome.storage.local. These settings stay local to the browser profile.

User Interface
  • User Interface Language: choose automatic language detection or a specific SBM UI language.
  • Theme: choose the visual theme used by SBM.
  • Details pane position: place the Details pane on the right side or below the Folder Contents pane.
  • Font: choose one of the supported font families.
  • Font size: adjust the base UI text size.
  • Font spacing: adjust the line spacing used by labels and rows.
Search

Limit Search to Current Folder & Subfolders changes the main search box from searching all bookmarks to searching only within the current folder tree.

Safety
  • Show delete confirmation warning: asks before deleting bookmark items or folders.
  • Show permanent sort confirmation warning: asks before applying permanent context-menu sorting.
  • Allow Delete key to delete selected items: controls whether the keyboard Delete key can remove selected items.
Sorting

Use natural folder/bookmark sort order controls the order used by permanent Sort by Name operations. When enabled, folders are placed first, separators next, then bookmarks.

Startup behavior
  • Allow multiple SBM instances: when disabled, clicking the extension button focuses the existing SBM tab instead of opening a new one.
  • Start at chosen folder: opens SBM at a selected bookmark folder when launched.
  • Startup folder: selects the folder used by the chosen-folder startup option.
Bookmark URL protections

These options control protections for higher-risk bookmark URL schemes when SBM opens bookmarks through extension tab or window actions. They do not prevent saving bookmarks.

  • Block opening javascript: bookmarks
  • Block opening data: bookmarks
  • Block opening blob: bookmarks
Pane widths
  • Library pane width: width in pixels for the left Library pane.
  • Details pane width: width in pixels for the right-side Details pane.
  • Bottom Details pane height: height in pixels for the bottom Details pane.

Pane sizes can also be changed by dragging the visible pane resize bars.

Folder Contents view

These options configure the optional columns and saved column widths in the middle Folder Contents pane. Name and URL are always shown.

  • Date Added: optional column showing when an item was added.
  • ID: optional column showing Chromium bookmark IDs.
  • Order: optional column showing Chromium folder order/index values.
Advanced Details
  • Enable advanced details viewing: shows read-only advanced bookmark metadata in the Details pane.
  • Enable advanced details editing: allows supported advanced editing, currently limited to bookmark index changes.
Optimisations

These options are mainly for benchmarking and troubleshooting. Leave them enabled unless you are comparing performance or diagnosing a rendering issue.

  • DOM rendering pass: reduces selected DOM churn in rendering paths.
  • Temporary bookmark tree maps: builds short-lived lookup maps during some tree operations, then discards them.
Debug Settings

Debug settings appear only when Debug options is enabled. The warnings/errors log is transient and is not saved to extension settings.
